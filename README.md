# datsnvim

## description

The datsnvim project is a neovim personal configuration for working with javascript, typescript and React.i

## installation

```bash
  git clone 'https://github.com/datsfilipe/datsnvim.git' $HOME/.config/nvim

  nvim +:PackSync
```
